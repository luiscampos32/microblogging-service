# Microblogging Service
A full-stack microblogging web application built with Python, Flask, Javascript, jQuery, and SQLite. Users can login/signup, edit profiles, follow other users, and create text/photo posts.

## Features
- User registration and authentication
- User profile creation and editing
- Create posts with text or images
- Follow and unfollow other users
- Personalized user feed
- View posts from followed users
- Responsive frontend with JavaScript interactivity

### Backend
- Python
- Flask
- SQLite
